## âœ¨ Features

* **User-Friendly GUI**: Built entirely with Tkinter and styled with `ttkthemes` for a clean, modern interface.
* **Multi-Threaded Performance**: Customize the number of concurrent threads (up to 500) to speed up your scanning process.
